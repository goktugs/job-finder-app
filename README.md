### Requirements

For development, you will need Node.js installed on your environement.
    
## Install

    $ git clone https://github.com/ORG/PROJECT.git
    $ cd PROJECT
    $ yarn

### Configure app

Copy `.env.example` to `.env`.

## Start 

    $ yarn dev
    
## To Run Tests

    $ yarn build

## Simple build for production
